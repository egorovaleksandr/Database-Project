 Предметная область:

В данном проекте рассматривается автосалон с точки зрения баз данных.
Выделенные основные сущности до нормализации:
1.	Клиент
2.	Продавец
3.	Автомобиль
4.	Услуга (продажа)
5.	Производитель
6.	Модель

База данных в 3НФ, т.к. нет необходимости в транзитивных зависимостях, также 2НФ очевидна.
